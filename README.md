Console Tic-Tac-Toe Game

This project is a console-based Tic-Tac-Toe game developed using Java. The game allows a human player to play against the computer on a 3×3 board. The application was built step-by-step by implementing different use cases such as board creation, user input handling, move validation, computer-generated moves, winner checking, and draw detection.

At the start of the game, a toss is performed to decide which player goes first and which symbol (X or O) each player receives. The human player enters a slot number between 1 and 9, and the program converts that slot into the correct row and column position on the board. The computer generates random valid moves during its turn.

The game continues in a loop until either a player wins or the match ends in a draw. Winning conditions are checked across rows, columns, and diagonals after every move.

Human vs Computer gameplay
