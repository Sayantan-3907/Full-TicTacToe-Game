# Console Tic-Tac-Toe Game

This is a console-based Tic-Tac-Toe game developed using Java where a human player competes against the computer.

## Features
- 3×3 Tic-Tac-Toe board
- Human vs Computer gameplay
- Toss to decide first player
- User move validation
- Random computer moves
- Win detection
- Draw detection
- Continuous game loop

## Concepts Used
- 2D Arrays
- Nested Loops
- Conditional Statements
- Methods
- Random Number Generation
- User Input Handling
- Boolean Logic

## How to Run

### Compile the program
```bash
javac TicTacToe.java
```

### Run the program
```bash
java TicTacToe
```
